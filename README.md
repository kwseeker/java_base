# java_base

Java语言及高级特性学习

## 工程目录

#### base_grammer
Java基础语法
+  use_enum: Java enum使用方法
+  access-right: 类和成员访问权限及单java文件多个平级类的限制
+  useAnotation: 自定义注解（编译期注解+RUNTIME注解）
+  super_class: super()方法默认执行父类的无参构造函数
+  useClass: Class类的使用

#### dynamic_proxy
java JDK及第三方库CGLIB实现的两种动态代理

#### socket_proj
java socket编程
